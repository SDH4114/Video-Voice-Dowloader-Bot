# Video-Voice-Dowloader-Bot

Телеграм-бот, который скачивает видео/аудио по ссылке (YouTube, TikTok, Instagram, Facebook, X, Pinterest и др.), конвертирует в совместимый MP4 и дополнительно высылает MP3-дорожку.

## Возможности
- Принимает ссылку и отправляет видео (поддержка стриминга) или документ, если файл крупный.
- Доставляет отдельный MP3 с аудио-дорожкой.
- Пытается выбирать совместимый H.264/AAC и при необходимости перекодирует через `ffmpeg`.
- Для Pinterest умеет подхватывать cookies из браузера/файла, если нужен вход.

## Требования
- Python 3.10+
- Установленный `ffmpeg` в PATH (нужен для перекодирования/извлечения аудио).
- Токен бота от BotFather.

## Установка
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Настройка окружения
Создайте файл `.env` (рядом с `main.py`) и задайте переменные:
```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
# Для Pinterest (опционально, если требуется авторизация):
# YTDLP_BROWSER_COOKIES=safari   # или chrome / brave / chromium
# YTDLP_COOKIE_FILE=/path/to/cookies.txt
```

## Запуск
```bash
source .venv/bin/activate
python main.py
```
В консоли появится `Bot started`. После этого бот работает в режиме long polling.

## Как использовать
1. Напишите боту команду `/start` или сразу отправьте ссылку на видео/аудио.
2. Дождитесь ответа «Скачиваю...», затем бот пришлёт видео и отдельный MP3.
3. Если файл больше ~50 МБ, бот отправит его как документ.

## Зачем это нужно
Быстрый способ сохранять ролики и аудио из популярных площадок прямо в Телеграм без лишних шагов и конвертации вручную.
